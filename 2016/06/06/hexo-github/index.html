Hexo + Github

从本地到github远程整个操作流程：

https://gist.github.com/yt8yt/ff97d36f7a35948a8351
其中遗漏了一个关键点，预先在github建立名为xxx.github.io的空工程。

主题

https://github.com/tufu9441/maupassant-hexo
感谢这位同学。

CNAME生成器

https://github.com/leecrossley/hexo-generator-cname
官方使用貌似有误，直接安装，然后hexo deploy -g便可生成CNAME文件，且CNAME域名是根据_config.yml中url所生成的。
例如http://techbirds.wang/ -> techbirds.wang

其实不用这个插件也挺方便，直接cd到source手动创建便可。

域名直接在阿里万网购买

阿里对域名解析设置说明特别清楚，我设置的是顶级域名(techbirds.wang)直接指向techbirds.github.io
